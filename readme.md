# Prompt Engineering Lab

## Overview

This project demonstrates practical prompt engineering techniques using the OpenAI API. The lab focuses on improving prompt reliability, consistency, and production-readiness through systematic testing and iterative refinement.

The project includes:
- Sentiment Analysis
- Product Description Generation
- Data Extraction

Techniques implemented:
- Zero-shot prompting
- Few-shot prompting
- Chain-of-Thought prompting
- Structured output constraints
- Failure analysis and consistency testing
