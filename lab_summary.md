# Lab Summary

In this lab, I explored how prompt engineering techniques improve the reliability and consistency of AI systems by testing sentiment analysis, product description generation, and data extraction prompts across multiple iterations. The initial zero-shot prompts produced inconsistent formatting, variable wording, and unstable outputs, especially in creative generation tasks. Through systematic testing with 5, 10, and 15 runs, I identified failure patterns and applied prompt engineering techniques including explicit constraints, structured formatting, few-shot prompting, and Chain-of-Thought reasoning. The most effective improvements came from adding deterministic output rules and structured schemas, which increased consistency from 60% to 100% for sentiment analysis and from 33.33% to 100% for data extraction. Product description generation remained linguistically variable due to the creative nature of the task, but formatting and structural consistency improved significantly. If I continued improving this system, I would experiment with function calling, stricter validation rules, and automated output evaluation to further stabilize generative tasks in production environments.
